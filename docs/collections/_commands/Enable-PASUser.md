# Enable-PASUser

## SYNOPSIS

Enables a specific vault user.

## SYNTAX

```
Enable-PASUser [-id] <Int32> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Reenables a disabled vault user

## EXAMPLES

### Example 1
```powershell
Enable-PASUser -id 1234
```

Enables the vault user with id 1234
