# Get-PASLoggedOnUser

## SYNOPSIS
Returns details of the logged on user

## SYNTAX

```
Get-PASLoggedOnUser [<CommonParameters>]
```

## DESCRIPTION
Returns information on the user who is logged in.

## EXAMPLES

### EXAMPLE 1
```
Get-PASLoggedOnUser
```

Returns information on the currently authenticated user.
