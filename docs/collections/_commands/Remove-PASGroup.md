# Remove-PASGroup

## SYNOPSIS
Deletes a user group

## SYNTAX

```
Remove-PASGroup [-GroupID] <Int32> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes a user group.

To delete a user group, the following authorizations are required:
- Add/Update Users

## EXAMPLES

### EXAMPLE 1
```
Delete-PASGroup -GroupID 3
```

Deletes Group with ID of 3
