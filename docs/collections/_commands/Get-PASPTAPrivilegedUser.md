# Get-PASPTAPrivilegedUser

## SYNOPSIS

## SYNTAX

```
Get-PASPTAPrivilegedUser [[-ValueType] <String>] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### Example 1
