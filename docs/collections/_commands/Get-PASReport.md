# Get-PASReport

## SYNOPSIS
Returns a list of available reports

## SYNTAX

```
Get-PASReport [<CommonParameters>]
```

## DESCRIPTION
Returns a list of reports available to the authenticated user

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-PASReport
```

Returns a list of all available reports
